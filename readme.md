# Name（リポジトリ/プロジェクト/OSSなどの名前）

Vanilla-SPA

# Detail

SPAをVanilla JSのみで実装

index.htmlに app を設置し、pathで判定して、該当のjsをrenderしています

お問い合わせは、送信データをMySQLに保存、自動返信を実装

メールサーバがなかったため、別サーバからiframeしています。

# Stack

* JavaScript
* firebase
* php
* MySQL
* sass
